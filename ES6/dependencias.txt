1) Para iniciar um projeto Yarn: yarn init

2) Para instalar a interface do bable: yarn add @bable/cli

3) Para instalar saber o ambiente(browser e etc): yarn add @babel/preset-env

4) Para executar: yarn dev(variavel criada no package.json).

--- Configurar o babel:

5) Criar o arquivo ".babelrc"

--- Instalação de Plugins:

6) Para instalar os operadores rest/spread: yarn add @babel/plugin-proposal-object-reset-spread
    Tem que colocar uma variavel no arquivo ".babelrc".
